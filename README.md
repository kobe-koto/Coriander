# Coriander

Coriander is a web-based tool for generating CSS gradients.

Preview: [https://coriander.pages.dev](https://coriander.pages.dev)

## Project Setup

This project is built with [Astro](https://astro.build/).

### Prerequisites

- [bun](https://bun.sh/)

Node.js and other package managers may work, but i didn't test them

### Development

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/kobe-koto/Coriander.git
    cd Coriander
    ```

2.  **Install dependencies:**
    ```bash
    bun install
    ```

3.  **Start the development server:**
    ```bash
    bun run dev
    ```
    The application will be available at `http://localhost:4321`.

### Build

To create a production build, run:
```bash
bun run build
```
The output will be in the `dist/` directory.
